# NodeWebMVCDemo

Faculty of Informatics ,Mahasarakham University

## Overview

A REST API . You will also learn how to use

- Postman
- MySQL
- Express.js
- Express-validator
- Sequelize ORM
- async/await

## API Endpoints

> GET /blog/list

List all Blogs

> POST /blog/create

Add new blog

> DELETE blog/delete/:id

Delete blog

> PUT /blog/update/:id

Update Blog
